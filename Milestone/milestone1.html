How Does IDAPro Detect Malware:

IDAPro is a disassembler/debugger program that is primarily used to retrieve and translate the machine code of software suspected to be nefarious in nature. The machine code is translated to assembly language, although there are techniques implemented into the program that can translate it into a more high-level language similar to the softwareâ€™s source code. A built-in debugger allows an analyst to examine the code without executing the program. This technique is allows the user to reverse engineer the malware safely without execting on the host machine.
